## What's Now Automatic

### 1. Sitemap Generation
- **Before**: Manual `sitemap.xml` file you had to update
- **After**: Auto-generated by `jekyll-sitemap` plugin

### 2. Related Posts
- **Before**: Manual "Related Articles" sections in each post
- **After**: Auto-generated based on matching categories and tags
- **Logic**: Matches posts with same category (2 points) + matching tags (1 point each)
- **Fallback**: Shows recent posts if no category/tag matches
- **You do**: Just add `category` and `tags` to frontmatter

### 3. Homepage Article Listings
- **Before**: Manually list every article on index.md
- **After**: Can use includes to auto-generate by category
- **Available includes**:

  - `{% include article-grid.html %}` - Grid layout of posts
- **You do**: Optionally replace manual lists with includes

### 4. RSS Feed (Bonus!)
- **New feature**: Auto-generated RSS feed at `/feed.xml`
- **Why**: Allows people to subscribe to your blog
- **You do**: Nothing, it's automatic

## How Related Posts Work Now

The post layout (`_layouts/post.html`) now automatically:
1. Finds posts with same category (+2 relevance points)
2. Finds posts with matching tags (+1 point per tag)
3. Sorts by relevance score
4. Shows top 4 most related posts
5. Falls back to recent posts if none match

**No more manual "Related Articles" sections needed!**

## Frontmatter Template for New Articles

When you create a new article, use this template:

```yaml
---
layout: post
title: "Your Article Title"
description: "150-160 char description for SEO"
keywords: "keyword1, keyword2, keyword3"
author: "Generalist Programmer"
category: "Technical Guides"  # or "Case Studies" or "Product Guides"
tags: ["tag1", "tag2", "tag3", "tag4"]  # For related posts matching
date: 2025-11-18
read_time: 10
modified: 2025-11-18
canonical_url: "https://generalistprogrammer.github.io/article-name.html"
robots: "index, follow"
---
```

**Important**: The `category` and `tags` fields are key for automatic related posts!
